# MQTT Emulator

Install MQTT.js:

`npm install mqtt`

Run emulator on vm (*10.3.69.32*):

`npm run vm`

Run emulator on local server (*127.0.0.1*):

`npm run local`

## screw-conveyer.js

螺旋输送机

* 环境温度：0-50
* 转速：30-60

## slurry-trunk.js

浆料箱

 * 箱内温度：0-60
 * 重量：0-1000

## centrifuge.js

离心机

 * 转速：600-1200
